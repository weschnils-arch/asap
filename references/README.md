# References

Design inspiration and competitor research.

**What goes here:**
- Screenshots of websites the client likes
- URLs to reference/competitor sites (put in a `urls.txt` file)
- Screenshots from the onboarding showing specific design ideas
- Mockups or wireframes (Figma exports, sketches, etc.)
- Pinterest boards or mood board exports

**Tips:**
- A simple `urls.txt` with one URL per line works — the agent will fetch and analyze them
- Annotate screenshots if specific elements are important (e.g. "client loved this nav style")
